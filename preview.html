How to run locally

  • Site only: open index.html directly, or run python3 -m http.server from the repo root.
  • Site + game in one server: run python3 serve.py from the repo root, then open http://localhost:8000/preview.html. The game lives at game/ with relative asset paths, so any static host works without rewrites.
  • To rebuild the game after editing game/src/components/CroncoreBlocks.tsx: cd game && ./build.sh (handles install + Vite build + bundle patch + promote to game/).
  • WebGPU is required. Chrome 121+, Edge 121+, Arc on desktop. Safari and Firefox stable do not work yet.
